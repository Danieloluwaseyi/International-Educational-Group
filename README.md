# International-Educational-Group
Power BI dashboard analyzing revenue, student enrollment, telc exams, and YoY growth across International School Locations.
‎# Project Deliverables
‎This project was completed and delivered with the following components:
‎
‎Fully Functional Power BI (.pbix) File with Source Mapping
‎
‎A complete Power BI report file containing:
‎
‎Cleaned and modeled data
‎
‎Interactive dashboards and visuals
‎
‎DAX measures for KPIs and trends
‎
‎Clear source mapping between Power BI tables and the original Google Sheets data
‎
‎The .pbix file is fully functional and can be opened locally in Power BI Desktop without any additional setup.
‎
#‎Documentation on Data Model and Update Process
‎
‎Comprehensive documentation covering:
‎Data model structure and relationships
‎
‎Tables and key calculated measures
‎
‎Refresh process when new data is added to the source
‎
‎Step-by-step instructions for manual updates using Power BI Desktop
‎
‎This ensures the report can be easily maintained and understood by other analysts or stakeholders.
‎
#‎Suggestions for Automation (Optional)
‎
‎Recommendations for optional automation, including:
‎
‎Publishing the report to Power BI Service
‎
‎Setting up scheduled refresh using Power BI Service
‎
‎Alternative automation options such as API connectors or Power Automate
‎
#‎Note:
‎Automation is not required for this project to function.
‎All analysis and visuals work fully in Power BI Desktop with manual refresh.
‎
‎These automation options are provided as future enhancement suggestions for production or enterprise environments.
‎
#‎ Summary
‎✔ Core objectives were achieved using Power BI Desktop
